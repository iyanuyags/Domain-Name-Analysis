# Domain-Name-Analysis

### Objective

To perform DNS enumeration on fezzant.com in order to identify DNS records, subdomains, and potential misconfigurations that could expose security risks.

### Skills Learned

- DNS enumeration and reconnaissance techniques
- Identification of DNS record types (A, AAAA, MX, NS, TXT)
- Subdomain discovery and analysis
- Understanding DNS-based attack surfaces
- Basic reconnaissance and information-gathering skills

### Tools Used

- dnsrecon / dig / nslookup – DNS enumeration and record analysis
- Kali Linux – Testing environment
- DNS Protocol Concepts – Domain name resolution and records

### Steps

- Selected fezzant.com as the target domain for enumeration.
- Enumerated DNS records to identify IP addresses, mail servers, and name servers.
- Performed subdomain discovery to uncover additional exposed services.
- Analyzed DNS configurations for potential misconfigurations.
- Documented discovered records and findings.
- Assessed how the information could be leveraged during reconnaissance or attacks.
